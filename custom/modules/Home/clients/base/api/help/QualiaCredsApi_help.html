Overview

A custom endpoint that save and reterive the data using config file. Returns the data in the config file of Integration URLs.

Path Variables

Name Description
:data It is an assosiative array that contains all related data of Integration Urls.

Input Parameters

This endpoint does not accept any input parameters.

Result

Name Type Description
integration_urls Assosiative Array Contains all the data in the integration array in the config file.
